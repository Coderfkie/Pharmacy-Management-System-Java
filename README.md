Pharmacy Management System (Java OOP Concept)

The main objectives of the Pharmacy Management System are:

To manage medicine records efficiently
To add, update, search, view, and sell medicines
To prevent selling expired or out-of-stock medicines
To apply Object-Oriented Programming concepts in a real-world system
To improve data organization using Java Collections
To provide a user-friendly menu-driven or GUI-based interface

Key OOP Concepts Used:

Encapsulation: Managing private data for medicines and sales.

Inheritance: Creating specialized classes for different types of users or products.

Polymorphism: Using method overriding for different functionalities.

Abstraction: Hiding complex logic using interfaces or abstract classes.
